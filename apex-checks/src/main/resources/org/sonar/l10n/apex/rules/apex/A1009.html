An async method, which is declared with an "@future" annotation, should not be called or invoked within a block of code inside a loop statement (for, while, do while).

Noncompliant Code Example

public with sharing class AsyncMethodClass {
        
        ...
        for(Accts newAcct : listAccts){
            ...
            newAcct.doSomething();
            execAsyncMethod();
            ...
        }
        ...
    
    @future
    public void execAsyncMethod(){
        ...
    }
}

Compliant Solution

public with sharing class AsyncMethodClass {
        
        ...
        for(Accts newAcct : listAccts){
            ...
            newAcct.doSomething();
            ...
        }
        execAsyncMethod();
        ...
    
    @future
    public void execAsyncMethod(){
        ...
    }
}
