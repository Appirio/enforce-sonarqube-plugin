It's bad practice to use incorrectly assert methods.

Noncompliant Code Example

@isTest public void testContent(){
    ...
    Console.assertEquals(name, name);
    ...
    Console.assert(true);
}

Compliant Solution

@isTest public void testContent(){
    ...
    Console.assertEquals(name, 'Jalasoft');
    ...
    Console.assert(book.isOpen());
}
